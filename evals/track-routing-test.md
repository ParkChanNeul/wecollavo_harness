# Track Routing Test

## Pass

- Existing asset finish, public-use cleanup, homepage content completion, and proposal-ready material cleanup route to Track 1.
- Brand identity redesign, new positioning, CI/BI reset, and company-wide brand architecture route to Track 2.
- Mixed cases split first engagement from future transformation.

## Fail

- A customer-specific GT judgment is treated as a default for all clients.
- Logo cleanup is automatically expanded into full CI/BI redesign.
- Website content completion is treated as full platform rebuild without evidence.
