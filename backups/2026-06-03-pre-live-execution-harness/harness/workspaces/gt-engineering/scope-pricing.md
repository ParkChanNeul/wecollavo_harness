# Scope And Pricing

Client: GT 엔지니어링

Status: 미팅 전 1차 가설. 최종 가격과 범위는 Human Linchpin Review 필요.

## First-Phase Scope

1차 착수 구간은 전체 제작물을 완성하는 단계가 아니라, 본 제작에 바로 사용할
수 있는 구조와 범위표를 만드는 단계다.

Included:

- 기존 브로셔/PPT/홈페이지/로고 자료 검토
- 공식 회사소개서 목차 및 페이지 구성안 작성
- 홈페이지 페이지별 콘텐츠 구조안 작성
- Framer 홈페이지 더미 데이터 교체 범위 정리
- 기존 로고 유지/리터칭/변경 범위 판단
- 고객사가 제공해야 할 자료 목록 작성
- WeCollavo가 직접 정리/작성해야 할 콘텐츠 범위 산정
- 본 제작 견적 기준표 작성

## Excluded Scope

- 전체 회사소개서 디자인 완성
- 홈페이지 전체 콘텐츠 적용 및 QA
- 신규 CI/BI 디자인
- 대량 카피라이팅
- 실적/인증/사진의 사실 검증 대행
- 임원/현장 이미지 고급 보정
- 무제한 수정
- 공개 가능성에 대한 법무 판단

## Price Decomposition

| Item | Scope | Price | Why it is needed |
| --- | --- | ---: | --- |
| 기존 자료 진단 | 브로셔, PPT, 홈페이지, 로고 자료 검토 | 30만~40만 원 | 현재 자료 중 유지/삭제/보완할 항목을 나누기 위해 필요 |
| 회사소개서 구조안 | 공식 회사소개서 목차와 페이지 구성안 | 40만~50만 원 | 본 제작 페이지 수와 난이도를 산정하기 위한 기준 |
| 홈페이지 콘텐츠 기준 | Framer 페이지별 더미 데이터 교체 범위 정리 | 30만~40만 원 | 홈페이지 마감이 단순 수정인지 구조 보완인지 판단 |
| 로고/CI 범위 판단 | 기존 로고 리터칭, 파일 정리, 신규 CI 여부 구분 | 20만~30만 원 | 신규 디자인과 기존 정리를 분리해 비용 리스크를 낮춤 |
| 본 제작 범위표 | 포함/제외/추가비/필요 자료/일정 기준 정리 | 30만~40만 원 | 본 제작 견적과 계약 조건의 근거 |

Recommended first-phase price: `150만~200만 원`

Positioning: 별도 견적비가 아니라 전체 제작 프로젝트에 포함되는 1차 착수금.

## Additional-Fee Triggers

| Trigger | Why it changes scope | Pricing rule |
| --- | --- | --- |
| 회사소개서 페이지 수 증가 | 페이지 수가 디자인, 카피, 자료 정리 시간을 직접 늘림 | 페이지당 8만~15만 원 기준 재산정 |
| 복잡 페이지 증가 | 실적표, 도표, 그룹 구조, 기술 설명은 일반 페이지보다 시간이 큼 | 난이도별 별도 산정 |
| 고객 자료 미정리 | WeCollavo가 자료를 직접 읽고 정리/작성해야 함 | 콘텐츠 정리/작성 비용 추가 |
| 홈페이지 구조 개편 | 단순 콘텐츠 교체가 아니라 메뉴/흐름/CTA 재설계가 필요 | 별도 협의 |
| 신규 CI/BI 요청 | 기존 로고 리터칭과 완전히 다른 작업 | 별도 견적 |
| 최종 승인 후 수정 | 승인 이후 변경은 일정과 작업량을 새로 발생시킴 | 별도 일정/비용 산정 |

## Schedule Assumptions

- 1차 착수 구간: 자료 수령 후 1~2주
- 본 제작: 범위 확정 후 5~8주 예상
- 자료 제공이 늦어지면 전체 일정도 뒤로 밀린다.

## Client Material Dependencies

- 기존 회사소개서/PPT 원본
- 기존 브로셔 인디자인/PDF/일러스트레이터 파일
- 로고/CI 원본 파일
- Framer 접근 권한
- 사업 분야별 공식 설명문
- 실제 프로젝트 리스트와 공개 가능 사진
- 면허, 인증, 등록증, 수상, 협력사 자료
- 문의 수신 담당자 정보

## Payment And Delivery Guardrails

- 1차 착수금 입금 후 착수
- 자료 제공 일정과 작업 일정은 함께 관리
- 1차 착수 산출물 승인 후 본 제작 견적 확정
- 최종 승인 후 추가 수정은 별도 일정/비용으로 산정
- 공개 가능한 자료인지에 대한 최종 판단은 고객 확인 필요
