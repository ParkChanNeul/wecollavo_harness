# Korean Connection Lesson System v2
## Culture → Grammar Tool → Real Use Loop

> 작성 목적: 4주 수업 데이터를 기반으로 기존 생존 회화형 커리큘럼을 개선하고, 이후 하네스가 생성해야 할 수업 구조의 기준을 문서화한다.  
> 핵심 전환: **상황 중심 수업**에서 **문화 기반 문법 도구 + 실제 사용 루프**로 이동한다.

---

## 0. 한 줄 정의

> **문법을 가르치는 것이 아니라, 문화가 문법으로 변하는 순간을 가르친다.**

한국어 수업의 핵심은 표현 암기가 아니다.  
외국인 학습자가 실제 한국인과 마주쳤을 때 “왜 이 상황에서 이렇게 말해야 하는지”를 이해하고, 그 문화적 판단을 문법 도구와 핵심 표현으로 연결해 실제 생활에서 쓰게 만드는 것이다.

---

## 1. 4주 수업 데이터에서 얻은 결론

### 1-1. 살아남은 표현

4주 동안 가장 많이 살아남은 표현은 반드시 수업안에 있던 표현만은 아니었다.

학생이 실제로 쓰려고 노력한 표현:

- 감사해요
- 괜찮아요
- 가주세요
- 수고하셨습니다
- 고생하셨습니다

특히 `수고하셨습니다`, `고생하셨습니다`는 사적으로 알려준 표현이었지만 학생이 필요성을 강하게 느꼈고, 실제로 쓰려고 노력했다.

### 1-2. 죽은 표현

거의 잊힌 표현:

- Week 3 카페/쇼핑 표현 다수
- Week 4 브이로그/스트리밍 오프닝 표현 다수

이유는 표현이 나빠서가 아니다.  
학생이 실제로 그 상황에 충분히 노출되지 않았기 때문이다.

### 1-3. 핵심 진단

```text
자주 마주침
+ 영어로 우회하기 어려움
+ 한국어를 쓰면 바로 반응이 옴
+ 짧고 외우기 쉬움
+ 문화적 의미가 분명함
= 살아남는 표현
```

반대로 아래 조건이면 표현은 죽는다.

```text
상황 노출이 적음
+ 영어/배달/앱으로 우회 가능
+ 실제 사용 압박 없음
+ 수업 후 체크 없음
= 다음 주에 사라짐
```

## 2. 기존 시스템의 강점과 한계

### 2-1. 유지해야 할 강점

기존 수업의 방향은 틀리지 않았다.

- 문화 차이와 해석 중심
- 실전 생존 회화 중심
- Role Play 중심
- Google Docs 치트시트
- 시각적 문법 구조
- SHOW IT / WAIT / BRIDGE / SPLIT

특히 첫 수업에서 학생이 `죄송합니다 + 한국말 잘 몰라요`의 순서를 바꾸거나, `-주세요`를 다른 표현에 응용한 것은 학생이 문장을 통째로 암기한 것이 아니라 구조를 이해했다는 증거다.

### 2-2. 개선해야 할 한계

기존 4주 수업은 “상황별 생존 표현”으로는 좋았지만, 성장 루프로는 부족했다.

문제는 다음과 같다.

1. 수업 밖 실제 사용 루프가 약했다.
2. 학생 자습과 앱 학습에 너무 많이 의존했다.
3. 수업 후 교재 공유와 숙제 체크가 약했다.
4. 노출 빈도가 낮은 상황을 너무 빨리 다뤘다.
5. 수업 중 교사가 말하는 비중이 높아졌다.
6. 문법 도구는 있었지만 문화와 문법의 연결 설명이 충분히 체계화되지 않았다.

## 3. 새 수업 철학

### 3-1. Situation-first에서 Culture-first로

기존 구조:

```text
상황
→ 표현
→ 롤플레이
→ 미션
```

개선 구조:

```text
문화적 긴장
→ 한국 사회 규칙
→ 사회적 의미
→ 언어 메커니즘
→ 문법 도구
→ 핵심 표현
→ 변형/예외
→ 롤플레이
→ 실제 미션
→ 다음 수업 체크
```

핵심은 “오늘 촬영장 표현을 배운다”가 아니다.

> **한국인은 일을 끝낼 때 그냥 떠나지 않고, 서로의 노력과 고생을 인정한다. 그래서 한국어에는 그 존중을 담는 표현과 말끝이 필요하다.**

이 문화적 초석이 먼저 깔려야 문법과 표현이 살아난다.

## 4. Culture → Grammar Tool → Real Use 구조

모든 수업은 아래 8단계로 설계한다.

### 1. Cultural Tension

외국인 학습자가 왜 이 상황에서 어색함을 느끼는지 정의한다.

예:

> “촬영이 끝났을 때 영어로는 그냥 Thanks, Bye라고 할 수 있지만, 한국에서는 그게 조금 차갑게 느껴질 수 있다.”

### 2. Korean Social Rule

한국인이 그 상황에서 기대하는 사회적 규칙을 설명한다.

예:

> “한국에서는 같이 일한 시간이 끝나면, 상대의 노력을 인정하고 관계를 부드럽게 닫는 말이 필요하다.”

### 3. Social Meaning

표현이 단순 번역이 아니라 어떤 감정/관계 기능을 하는지 설명한다.

예:

> `수고하셨습니다`는 단순한 goodbye가 아니다.  
> “I saw your effort. I respect the work you did today.”에 가깝다.

### 4. Language Mechanism

그 문화적 의미가 한국어 안에서 어떻게 작동하는지 보여준다.

예:

> 한국어는 상대의 노력과 지위를 말끝에 반영한다.  
> 그래서 `-시-`, `-셨-`, `-습니다` 같은 장치가 필요하다.

### 5. Grammar Tool

오늘의 문법 도구를 명확히 제시한다.

예:

```text
-시-        = respect
-었-        = completed action / past
-습니다     = formal ending

-시- + -었- + -습니다
= respectful completed formal ending
```

### 6. Core Expressions

실제로 바로 쓸 수 있는 핵심 표현을 제시한다.

예:

- 오늘 잘 부탁드립니다.
- 수고하셨습니다.
- 고생하셨습니다.
- 감사합니다.

### 7. Variation / Exception Box

학생이 반드시 헷갈릴 표현과 예외를 미리 정리한다.

예:

- 수고하셨습니다
- 수고하셨어요
- 수고했어요
- 수고했어

### 8. Real Use Mission

72시간 안에 실제 상황에서 쓰게 만든다.

예:

> 이번 주 촬영장, PT, 식당 협찬 촬영 중 하나에서 `수고하셨습니다`를 한 번 말하고 반응을 기록한다.

## 5. 기존 4대 원칙 업데이트

기존 4대 원칙은 유지한다.

```text
SHOW IT   → 구조를 시각적으로 보여준다
WAIT      → 학생이 생각하고 말할 시간을 준다
BRIDGE    → 단계 전환을 명확히 안내한다
SPLIT     → 한국어는 연습, 영어는 설명으로 분리한다
```

여기에 두 가지를 추가한다.

```text
USE IT    → 72시간 안에 실제 상황에서 쓰게 한다
CHECK IT  → 다음 수업 첫 10분에 사용 여부를 확인한다
```

따라서 v2의 원칙은 다음 6개다.

## The Six Teaching Rules

1. **SHOW IT** — 문화와 문법 구조를 시각화한다.
2. **WAIT** — 학생이 직접 문장을 만들 시간을 준다.
3. **BRIDGE** — 문화 → 문법 → 표현 → 사용으로 자연스럽게 연결한다.
4. **SPLIT** — 한국어는 연습, 영어는 설명으로 분리한다.
5. **USE IT** — 수업 후 실제 생활에서 반드시 한 번 쓰게 한다.
6. **CHECK IT** — 다음 수업에서 사용 여부와 반응을 반드시 확인한다.

## 6. 표준 수업 구조 v2

### 60분 수업 기준

```text
[0-10분]   Cold Recall + Mission Check
           - 지난주 표현을 힌트 없이 말하게 함
           - 실제로 썼는지 확인
           - 썼다면 반응 기록
           - 못 썼다면 이유 확인

[10-18분]  Cultural Tension
           - 오늘 상황에서 외국인이 느끼는 어색함 제시
           - 영어권 방식과 한국식 방식 비교

[18-25분]  Korean Social Rule
           - 한국인이 이 상황에서 기대하는 감정/예절/관계 규칙 설명
           - 단순 표현이 아니라 사회적 의미 설명

[25-35분]  Grammar Tool
           - 오늘의 문법 도구 1개만 명확히 제시
           - 색상 구조로 분해
           - 문법은 문화적 의미와 연결해서 설명

[35-43분]  Core Expressions + Variation Box
           - 핵심 표현 3~5개
           - 헷갈리는 변형과 예외 정리

[43-53분]  Role Play
           - 대표 상황에서 학생이 70% 이상 말하기
           - 교정은 끝까지 듣고 난 뒤

[53-58분]  Real Mission Contract
           - 언제, 어디서, 누구에게 쓸지 정함
           - 학생이 직접 문장을 선택함

[58-60분]  Phone Setup
           - Google Docs / 메모장 / 메시지로 바로 열 수 있게 저장
```

## 7. Week 5 예시 문서

## Week 5: Korean for Work Moments
### How to End a Shoot Like Koreans Do

---

### 7-1. Cultural Thesis

> **한국인은 일을 끝낼 때 ‘결과’보다 ‘고생’을 먼저 인정한다.**

촬영장, PT, 식당 협찬 촬영, 수업, 업무 미팅 등에서 한국인은 단순히 `Bye`라고 하지 않는다.

같이 시간을 쓰고, 에너지를 쓰고, 어떤 일을 마쳤다면 그 끝에는 상대의 노력을 인정하는 말이 필요하다.

---

### 7-2. Cultural Explanation

영어권에서는 일을 끝낼 때 보통 이렇게 말할 수 있다.

```text
Thanks!
Good job!
See you!
Bye!
```

하지만 한국어에서는 그 순간이 조금 다르게 작동한다.

한국식 감각은 다음에 가깝다.

```text
오늘 같이 일했다.
너도 에너지를 썼다.
나도 그걸 봤다.
그래서 그냥 떠나지 않고, 네 노력을 인정하고 간다.
```

그래서 `수고하셨습니다`는 단순한 끝인사가 아니다.

> “I saw your effort.”  
> “I respect the work you did.”  
> “Thank you for spending your energy today.”

에 가까운 관계 정리 표현이다.

---

### 7-3. Language Mechanism

이 문화는 한국어에서 말끝으로 표현된다.

한국어는 단어 자체보다도 **누구에게 말하는지**, **얼마나 정중해야 하는지**, **상대의 행동을 높여야 하는지**가 중요하다.

그래서 다음 장치가 필요하다.

```text
-시-      = respect
-었-      = completed / past
-습니다   = formal ending
```

즉:

```text
하 + 시 + 었 + 습니다
= do + respect + completed + formal
= 하셨습니다
```

---

### 7-4. Today's Grammar Tool

## Respectful Past Formal

```text
-시- + -었- + -습니다
```

### 비교

```text
했어요
= normal polite past
= I did / someone did

하셨어요
= respectful polite past
= someone respected did

하셨습니다
= respectful formal past
= formal work-ending version
```

---

### 7-5. Core Expressions

### 1. 오늘 잘 부탁드립니다.

```text
오늘       잘        부탁드립니다
Today     well      please take care of me / I look forward to working with you
```

사용 상황:

- 촬영 시작 전
- PT 시작 전
- 협찬 식당 촬영 시작 전
- 새로운 스태프를 만났을 때

문화 의미:

> “I respect this situation, and I’m ready to work with you.”

---

### 2. 수고하셨습니다.

```text
수고      하      시      었      습니다
Effort    do     respect  past    formal
```

사용 상황:

- 촬영 끝난 뒤
- PT 끝난 뒤
- 수업 끝난 뒤
- 스태프/감독/선생님/트레이너에게

문화 의미:

> “I recognize your effort today.”

---

### 3. 고생하셨습니다.

```text
고생      하      시      었      습니다
Hardship  do     respect  past    formal
```

사용 상황:

- 상대가 힘들게 일했을 때
- 긴 촬영이 끝났을 때
- 날씨가 덥거나 일정이 빡빡했을 때

문화 의미:

> `수고하셨습니다`보다 조금 더 따뜻하고 감정적이다.  
> “You went through a lot today.”에 가깝다.

---

### 4. 감사합니다.

사용 상황:

- 거의 모든 상황
- 안전한 기본 표현

문화 의미:

> 감사 표현은 좋지만, 촬영장/업무 상황에서는 `수고하셨습니다`와 함께 쓰면 훨씬 자연스럽다.

예:

```text
수고하셨습니다. 감사합니다.
```

## 8. Variation / Exception Box

### 8-1. 수고하셨습니다 vs 수고하셨어요

```text
수고하셨습니다
= formal / safest / work setting
= director, staff, trainer, photographer

수고하셨어요
= softer / friendly polite
= someone you know better
= casual work setting
```

### 8-2. 수고하셨어요 vs 수고했어요

```text
수고하셨어요
= respectful
= for someone older, higher, or professional

수고했어요
= polite but no honorific
= close friend, younger person, equal relationship
```

### 8-3. 수고했어요 vs 수고했어

```text
수고했어요
= polite casual

수고했어
= casual
= close friend only
```

### 8-4. 자기 자신에게는 보통 쓰지 않음

```text
❌ 저는 수고하셨습니다.
❌ 저는 고생하셨습니다.

✅ 오늘 촬영했어요.
✅ 오늘 열심히 했어요.
✅ 오늘 조금 힘들었어요.
```

### 8-5. 처음에는 가장 안전한 표현만 쓴다

초급자는 아래 두 개만 먼저 입에 붙이면 된다.

```text
시작할 때:
오늘 잘 부탁드립니다.

끝날 때:
수고하셨습니다. 감사합니다.
```

## 11. Real Mission

### 이번 주 미션

```text
Say “수고하셨습니다” once this week.
```

### Mission Contract

```text
I will say: ______________________
To: ______________________________
Where: ___________________________
When: ____________________________
```

예:

```text
I will say: 수고하셨습니다. 감사합니다.
To: PT trainer
Where: Gym
When: After PT session
```

---

## 12. Mission-as-Content Loop

실제 미션은 단순 숙제가 아니다.  
학생의 학습 강제성, 성취감, 콘텐츠 아이디어, 튜터 마케팅을 동시에 만들 수 있다.

### 12-1. 3단계 미션 구조

```text
Level 1: Private Mission
- 학생이 실제로 말하고 튜터에게 메시지만 보냄

Level 2: Proof Mission
- 학생이 음성메모, 캡처, 짧은 후기 공유

Level 3: Content Mission
- 학생 동의하에 숏폼/후기/수업 사례로 활용
```

### 12-2. Tee-style Content Idea

```text
POV: My Korean mentor taught me the most useful phrase on set.

수고하셨습니다
= not just “good job”
= “I respect the effort you gave today.”
```

### 12-3. Consent Rule

수업 미션과 마케팅 콘텐츠는 반드시 분리한다.

```text
Mission = required for learning
Content = optional and consent-based
```

학생이 동의하지 않으면 미션은 비공개 학습 기록으로만 남긴다.

## 13. 하네스 생성 규칙

하네스는 단순히 PPT를 생성하면 안 된다.  
하네스는 **문화 기반 수업 운영 패키지**를 생성해야 한다.

### 13-1. Harness Input

```text
student_profile
real_life_exposure
recent_successful_expressions
forgotten_expressions
grammar_tool
cultural_tension
representative_situation
mission_context
content_consent_level
```

### 13-2. Harness Output

```text
1. Teacher Lesson Plan
2. Student PPT
3. Grammar Tool Card
4. Cultural Explanation Card
5. Variation / Exception Box
6. Google Docs Cheat Sheet
7. Real Mission Card
8. Follow-up Message
9. Content Mission Brief
10. Next Lesson Check Script
```

### 13-3. Harness Invariants

하네스가 반드시 지켜야 할 규칙:

```text
1. Grammar-first lesson is forbidden.
2. Every lesson must start from cultural tension.
3. Every grammar tool must explain social meaning.
4. Every lesson must include a real mission.
5. Every lesson must include next-week check.
6. Student speaking time must be at least 70% in role play.
7. Teacher explanation must not replace student production.
8. Every lesson must include variation/exception box.
9. Every lesson must produce a phone-ready cheat sheet.
10. Content use requires explicit student consent.
```

## 14. KPI

앞으로 수업의 성과는 “수업을 잘했다”가 아니라 실제 사용과 기억으로 판단한다.

### 14-1. Learning KPI

```text
Mission Completion Rate
- 이번 주 실제 미션을 수행했는가?

Spontaneous Use Count
- 학생이 먼저 한국어를 쓰려고 했는가?

Retention Check
- 다음 주에 힌트 없이 말할 수 있는가?

Variation Understanding
- 수고하셨습니다 / 수고하셨어요 차이를 이해하는가?
```

### 14-2. Business KPI

```text
Rebooking Rate
- 다음 수업으로 이어졌는가?

Review / Testimonial
- 학생이 수업 효과를 말로 남겼는가?

Content Proof
- 동의 기반 콘텐츠 자산이 생겼는가?

Referral Potential
- 학생이 친구나 팔로워에게 공유할 이유가 생겼는가?
```
