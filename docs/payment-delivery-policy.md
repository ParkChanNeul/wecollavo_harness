# Payment And Delivery Policy

## Payment Terms

| Project Amount | Payment Terms |
|----------------|---------------|
| 500만 원 미만 | 선결제 100% |
| 500만~1,000만 원 미만 | 계약금 50% / 최종 납품 전 잔금 50% |
| 1,000만 원 이상 | 계약금 10% + 착수금/중도금/잔금 단계별 협의 |
| 외부 솔루션 비용 | 고객 부담 |
| 긴급 일정 | 가능 여부 확인 후 추가비 적용 |

## Delivery Conditions

- 최종 산출물, 소스코드, 관리자 권한, Framer 권한, 인쇄용 파일은 잔금 확인 후 전달한다.
- 파일 보관 기본 기간은 30일이다.
- 납품 후 경미 수정 기준 기간은 14일이다.
- 긴급 일정은 가능 여부 확인 후 추가비를 적용한다.

## Client-Safe Phrase

최종 산출물, 소스코드, 관리자 권한, Framer 권한, 인쇄용 파일 등은 잔금 확인 후 전달됩니다.
