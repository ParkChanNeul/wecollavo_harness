# Delivery OS

The proposal must become delivery operations.

```text
Proposal Data
  -> Execution Asset Brief
  -> Delivery Plan
  -> Task List
  -> Review Checklist
  -> Client Delivery
  -> Proof Loop
```

`delivery-plan.json` must include:

- approved scope
- excluded scope
- client dependencies
- tasks
- review gates
- delivery checklist
- scope-change rules

The first-phase scope from the proposal becomes the first task group.
