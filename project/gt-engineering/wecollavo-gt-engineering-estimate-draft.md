# GT 엔지니어링 제안/견적 초안

Client: GT 엔지니어링
Industry: 토목/건설, 시설안전관리, 종합건축, 해양 수리조선
Product: 브랜드 자산 마감 스프린트
Status: Draft

## 1. 제안 요약

GT 엔지니어링은 토목/건설, 시설안전관리, 종합건축, 해양 수리조선까지 복수 사업 영역을 운영하는 그룹사입니다.

현재 필요한 작업은 단순 브로셔 재디자인이 아니라, 흩어져 있는 회사 자료와 기존 제작물을 바탕으로 **공식 외부 제출용 회사소개서, 홈페이지, 로고/CI 적용 기준을 다시 정리하고 마감하는 작업**입니다.

기존에 제작된 브로셔와 로고 프로토타입은 출발점으로 활용하되, 고객, 제휴사, 투자자, 관공서/입찰, 내부 영업팀이 함께 볼 수 있는 공식 회사소개 자료로 재구성하는 것을 목표로 합니다.

## 2. 현재 확인된 상황

- 기존 PPT 콘텐츠를 참고해 신규 인디자인/PDF 및 일러스트레이터 기반 브로셔를 납품한 이력이 있음
- 기존 납품 브로셔는 내부 참고용으로는 활용 가능하나, 공식 회사소개서로는 재디자인 필요
- 홈페이지는 기존 콘텐츠를 전달받아 마감해야 하는 상태
- 홈페이지 납품 링크는 https://gotogether.framer.website 이며, 사업 영역과 카테고리 구조를 제외하면 대부분 더미 데이터로 봐야 함
- 회사 자료가 정리되어 있지 않아 콘텐츠 수급 병목이 장기적으로 발생
- 브로셔 제작 당시 CI/BI 프로토타입을 제작했고, 고객 반응은 긍정적
- CI/BI 변경 또는 신규 디자인은 별도 비용 발생 가능
- 간부 프로필 사진 등 공개용 이미지 품질이 낮을 가능성이 있음
- 고객은 전체 비용을 낮게 유지하고 싶어하지만, 요구 범위가 넓어질 가능성이 큼

## 3. 핵심 리스크

이번 작업의 가장 큰 리스크는 디자인 난이도보다 **콘텐츠 정리와 의사결정 지연**입니다.

공식 회사소개서와 홈페이지를 만들려면 아래 정보가 필요합니다.

- 사업 영역별 공식 설명
- 주요 실적과 프로젝트 정보
- 인허가, 인증, 면허, 수상, 협력사 등 신뢰 자료
- 조직/그룹사 구조
- 대표 및 주요 임원 프로필
- 고객/제휴/입찰용으로 공개 가능한 사진과 이미지
- 홈페이지에 게시 가능한 최종 문구

자료가 정리되어 전달되면 제작비는 낮아질 수 있습니다.

반대로 WeCollavo가 회사 자료를 열람하고 직접 내용을 정리해야 한다면, 이는 단순 디자인 작업이 아니라 **콘텐츠 기획/작성/편집 업무**가 포함되므로 비용과 일정이 증가합니다.

## 4. 권장 진행 방식

이번 건은 견적을 따로 유료로 산정하는 방식이 아니라, **전체 제작 프로젝트를 1차 착수 구간과 본 제작 구간으로 나누어 진행**하는 방식이 적합합니다.

고객에게는 전체 예상 범위를 먼저 제시하되, 첫 착수 구간에서 공식 회사소개서의 목차, 홈페이지 콘텐츠 기준, 로고 적용 범위, 우선 제작 페이지를 실제로 정리합니다.

```text
1단계: 공식 회사소개서/홈페이지 착수 기획
2단계: 공식 회사소개서 재디자인
3단계: 홈페이지 콘텐츠 마감
4단계: 로고/CI 리터칭 및 적용 정리
5단계: 공개용 이미지/프로필 보정 여부 결정
```

## 5. 1단계 제안: 공식 회사소개서/홈페이지 착수 기획

### 목적

제작 전에 단순히 견적만 산정하는 것이 아니라, 본 제작에 바로 사용할 수 있는 공식 회사소개서 구조와 홈페이지 콘텐츠 기준을 먼저 만듭니다.

### 작업 내용

- 기존 브로셔/PPT/홈페이지/로고 파일 검토
- 공식 회사소개서 목차 및 페이지 구성안 작성
- 홈페이지 페이지별 콘텐츠 구조안 작성
- Framer 홈페이지 내 더미 데이터 교체 범위 정리
- CI/BI 유지, 리터칭, 변경 범위 판단
- 간부 프로필 사진 및 공개 이미지 사용 가능성 검토
- 고객사가 제공해야 할 자료 목록 작성
- WeCollavo가 직접 작성/정리해야 하는 콘텐츠 범위 산정
- 우선 제작 페이지와 후순위 페이지 구분
- 본 제작 비용 산정을 위한 작업 범위표 작성

### 산출물

- 공식 회사소개서 구성안
- 홈페이지 콘텐츠 구조안
- 로고/CI 작업 범위표
- 추가 콘텐츠 작성 필요 항목
- 우선 제작 페이지 목록
- 본 제작 견적서

### 1단계 착수금

**150만~200만 원**

이 금액은 전체 제작 프로젝트에 포함되는 **1차 착수금**입니다.

1단계에서 만든 회사소개서 구성안, 홈페이지 콘텐츠 구조안, 로고 작업 범위표는 이후 본 제작에 그대로 사용합니다. 본 제작까지 이어질 경우, 1단계 산출물은 별도 폐기되는 문서가 아니라 전체 제작물의 기획/구조 파트로 포함됩니다.

## 6. 본 제작 예상 범위

아래 비용은 1단계 착수 기획 이후 페이지 수와 자료 제공 수준에 따라 확정합니다.

### 공식 회사소개서

기존 브로셔 기준 40p 내외였으나, 공식 회사소개서가 50p 이상으로 확장될 가능성이 있습니다.

| 구분 | 기준 | 단가 |
|------|------|------|
| 일반 페이지 | 텍스트와 이미지 중심 구성 | 8만 원 / p |
| 복잡 페이지 | 사업 구조, 실적, 표, 도표 포함 | 12만~15만 원 / p |
| 고난도 페이지 | 그룹사 구조, 기술/입찰/투자 설득 구조 포함 | 별도 협의 |

예상 범위:

- 40p 기준: 약 320만~600만 원
- 50p 기준: 약 400만~750만 원

페이지 난이도, 자료 정리 수준, 신규 콘텐츠 작성 여부에 따라 변동됩니다.

### 홈페이지 마감

기존 기준 홈페이지는 15p 내외로 예상됩니다.

현재 홈페이지는 Framer로 납품된 상태이며, 사업 영역과 카테고리 구조를 제외하면 더미 데이터가 다수 포함되어 있습니다. 따라서 홈페이지 마감은 단순 적용 작업이 아니라, 실제 회사 정보, 프로젝트 사례, 이미지, 문의 정보를 공개 가능한 콘텐츠로 교체하는 작업입니다.

| 구분 | 기준 | 금액 |
|------|------|------|
| 경량 마감 | 전달받은 콘텐츠를 기준으로 문구/이미지/섹션 정리 | 80만~150만 원 |
| 일반 마감 | 페이지별 콘텐츠 정리와 구성 보완 포함 | 150만~300만 원 |
| 구조 개편 | 메뉴/페이지 구조, CTA, 사업별 흐름 재설계 | 별도 협의 |

### 로고/CI/BI

기존 프로토타입 로고를 유지하고 리터칭하는 경우와, CI/BI를 변경하는 경우는 비용이 다릅니다.

| 구분 | 기준 | 비용 |
|------|------|------|
| 기존 로고 리터칭 | 형태 유지, 비율/간격/파일 정리 | 별도 산정 |
| CI 적용 정리 | 회사소개서/홈페이지 적용 기준 정리 | 별도 산정 |
| CI/BI 변경 | 신규 방향성, 응용 시스템 포함 | 별도 견적 |

### 프로필/이미지 보정

간부 프로필 사진, 현장 사진, 장비 사진 등 공개용 이미지 품질이 낮은 경우 별도 보정 또는 AI 기반 이미지 개선이 필요할 수 있습니다.

단, AI 이미지 생성/보정은 실제 인물과 회사 신뢰에 직접 영향을 주므로 사전 승인된 범위 안에서만 진행합니다.

## 7. 일정 제안

고객사가 현재 제작 범위와 자료 상태를 명확히 알기 어려운 상황이므로, 일정은 WeCollavo가 선제안하는 방식이 적합합니다.

### 권장 일정

```text
1주차: 자료 수령, 기존 제작물 검토, 회사소개서/홈페이지 구조 정리
2주차: 회사소개서 목차/홈페이지 콘텐츠 구조안/로고 범위 확정
3~5주차: 공식 회사소개서 디자인 및 콘텐츠 제작
6~7주차: 홈페이지 콘텐츠 마감 및 적용
8주차: 최종 수정, 파일 정리, 납품
```

자료 제공이 지연되거나 WeCollavo가 콘텐츠를 직접 정리/작성해야 하는 경우 일정은 연장됩니다.

## 9. 고객 전달용 핵심 문구

```text
이번 작업은 단순히 브로셔를 예쁘게 다시 만드는 작업이 아니라, GT 엔지니어링의 여러 사업 영역을 외부 제출 가능한 공식 회사소개서와 홈페이지로 정리하는 작업입니다.

현재는 자료가 흩어져 있고 공개 가능한 콘텐츠 기준이 명확하지 않아, 바로 전체 제작비를 확정하면 중간에 범위와 비용이 계속 흔들릴 가능성이 큽니다.

그래서 전체 제작을 한 번에 무리하게 확정하기보다, 먼저 1차 착수 구간에서 기존 브로셔, 홈페이지, 로고, 회사 자료를 검토해 공식 회사소개서 목차와 홈페이지 콘텐츠 구조안, 로고 작업 범위를 실제 제작 가능한 형태로 정리하는 것을 제안드립니다.

1차 착수금은 150만~200만 원이며, 이 금액은 전체 제작 프로젝트에 포함되는 기획/구조화 작업비입니다. 이 단계가 끝나면 실제 페이지 수, 콘텐츠 작성 범위, 홈페이지 마감 범위에 맞춰 본 제작 견적을 명확히 산정할 수 있습니다.
```

## 10. 견적 방향

현재 고객의 예산 감각은 150만 원 전후로 보이나, 요청 범위는 공식 회사소개서 40~50p 이상, 홈페이지 15p 내외, 로고 리터칭, 이미지 보정 가능성까지 포함하고 있어 전체 제작비로는 맞추기 어렵습니다.

따라서 150만 원 전후를 "견적비"로 제안하면 거부감이 큽니다. 대신 **전체 제작 프로젝트의 1차 착수금**으로 제안하는 것이 적합합니다.

핵심 표현은 "견적을 내기 위한 비용"이 아니라 "공식 회사소개서와 홈페이지 제작을 시작하기 위한 첫 구간"입니다.

최종 제작비는 아래 요소에 따라 별도 산정합니다.

- 회사소개서 최종 페이지 수
- 페이지별 디자인 난이도
- 고객사가 제공하는 콘텐츠의 완성도
- WeCollavo가 직접 작성/정리해야 하는 콘텐츠 범위
- 홈페이지 적용 범위
- 로고/CI/BI 변경 범위
- 프로필/이미지 보정 범위
