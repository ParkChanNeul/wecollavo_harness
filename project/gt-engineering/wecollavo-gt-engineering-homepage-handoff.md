## 1. 현재 상태

홈페이지는 Framer로 납품된 상태이며, 현재 링크는 아래와 같습니다.

https://gotogether.framer.website

현재 기준으로 사업 영역과 카테고리 구조를 제외한 대부분의 상세 콘텐츠는 더미 데이터로 봐야 합니다. 따라서 이번 홈페이지 마감 작업은 디자인 수정만이 아니라, **더미 데이터를 실제 공개 가능한 회사 콘텐츠로 교체하는 작업**입니다.

검색 인덱스 기준으로 확인되는 주요 구조는 아래와 같습니다.

- 메인
- 회사 소개
- 사업 분야
- 사업 분야 상세: 안전진단사업, 시설물유지보수, 종합 건설/개발, 수리조선 및 무역, 해상 레저, 프롭테크 플랫폼
- 프로젝트
- 프로젝트 상세
- 공사 현황/갤러리
- 블로그/인사이트
- 문의하기
- 개인정보처리방침
- 이용약관
- 스타일 가이드

## 2. 마감 기준

홈페이지 마감의 기준은 "페이지가 존재한다"가 아니라, 외부 고객, 제휴사, 투자자, 관공서/입찰 담당자에게 보여도 문제가 없는 실제 콘텐츠가 들어가는 것입니다.

따라서 아래 세 가지를 기준으로 마감 범위를 판단합니다.

1. 공개 가능한 실제 회사 정보인가
2. 회사소개서와 같은 메시지 체계로 연결되는가
3. 고객 문의, 제휴, 입찰, 영업에 사용할 수 있는 신뢰 자료인가

## 3. 우선 교체가 필요한 콘텐츠

### A. 회사 소개

필요 자료:

- GT 그룹 공식 소개문
- 설립 배경 또는 사업 전개 흐름
- 그룹사/계열사 구조
- 대표 인사말 사용 여부
- 조직도 또는 핵심 인력 소개 여부
- 면허, 인증, 등록 현황
- 주요 고객/협력사 공개 가능 여부

### B. 프로젝트

현재 프로젝트 상세에는 해외 건축/부동산 더미 사례가 섞여 있는 것으로 보입니다.

필요 자료:

- 실제 프로젝트명
- 발주처 또는 공개 가능한 고객명
- 위치
- 기간
- 사업 분야
- 수행 범위
- 공사/점검/용역 규모
- 진행 상태
- 사용 가능한 사진
- 공개 가능 여부

프로젝트는 모든 사례를 한 번에 채우기보다, 먼저 공개 가능한 대표 사례 5~8개를 선정하는 것이 좋습니다.

### C. 공사 현황/갤러리

필요 자료:

- 실제 현장 사진
- 사진별 현장명
- 촬영 시기
- 공개 가능 여부
- 사진 설명

사진 품질이 낮거나 휴대폰 촬영본만 있는 경우, 공개용 보정 또는 재구성이 필요할 수 있습니다.

### D. 블로그/인사이트

현재 블로그는 더미 콘텐츠로 보는 것이 안전합니다.

선택지는 두 가지입니다.

- 블로그 섹션을 1차 마감 범위에서 제외
- 실제 글 3~5개만 남기고 나머지는 비공개 처리

GT의 현재 상황에서는 블로그를 억지로 운영하기보다, 회사소개서/프로젝트/사업 분야 마감이 우선입니다.

### E. 문의하기

필요 확인:

- 문의 수신 이메일
- 문의 담당자
- 전화번호
- 주소
- 지도 연결
- 프로젝트 타입 선택 항목
- 개인정보 수집 동의 문구
- Framer 폼 수신 테스트

## 5. 고객에게 해야 할 질문

1. 현재 홈페이지에서 그대로 유지해도 되는 페이지는 무엇인가요?
2. 사업 분야 구조는 현재 상태로 확정인가요?
3. 프로젝트 페이지에 공개 가능한 실제 사례는 몇 개인가요?
4. 공개하면 안 되는 고객명, 현장명, 사진이 있나요?
5. 블로그/인사이트는 실제로 운영할 계획이 있나요, 아니면 숨기는 것이 나은가요?
6. 갤러리 사진은 누가 선별하나요?
7. 문의 폼은 실제로 누가 받나요?
8. Framer 관리 권한을 제공할 수 있나요?
9. 홈페이지와 회사소개서 중 어떤 자료를 먼저 외부에 공개해야 하나요?
10. 최종 오픈 목표일은 언제로 잡을까요?

## 6. 견적 반영 기준

홈페이지 마감 비용은 단순 페이지 수가 아니라, 더미 콘텐츠를 실제 콘텐츠로 교체하는 난이도에 따라 달라집니다.

| 구분 | 기준 | 견적 영향 |
|------|------|-----------|
| 단순 교체 | 고객이 최종 문구와 이미지를 제공 | 낮음 |
| 구조 보완 | 기존 페이지 구조는 유지하되 섹션 흐름 수정 | 중간 |
| 콘텐츠 작성 | WeCollavo가 회사 자료를 보고 문구 작성 | 높음 |
| 프로젝트 정리 | 실제 사례를 선별하고 설명문 작성 | 높음 |
| 이미지 보정 | 현장/임원/장비 사진 보정 또는 재구성 | 별도 |
| Framer 적용 | 실제 페이지에 콘텐츠 반영 및 반응형 확인 | 중간~높음 |

## 7. 제안 문구

```text
현재 홈페이지는 구조와 디자인은 잡혀 있지만, 사업 영역과 카테고리 외에는 더미 데이터가 많이 남아 있는 상태입니다.

따라서 이번 홈페이지 마감은 단순 수정이 아니라, 더미 데이터를 실제 공개 가능한 회사 정보, 프로젝트 사례, 사진, 문의 정보로 교체하는 작업입니다.

고객사에서 최종 문구와 이미지를 제공해주시면 적용 범위 중심으로 진행할 수 있고, 자료가 정리되어 있지 않다면 WeCollavo가 회사 자료를 열람해 콘텐츠를 직접 정리해야 하므로 비용과 일정이 추가됩니다.
```
